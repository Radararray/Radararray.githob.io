Radar on a Board

Abstract

The main goal of this project is to utilize and develop a radar that can detect objects in a 2D plane and improve on its current capabilities. The board utilizes a phased array antenna that detects and tracks objects in a 2D plane.

Project Background

As the use of technology has been expanding, radars have found themselves in many devices out there, thus making radars crucial components within the electronics industry. Phase array antennas like the one in the project are used in cars for cruise control, 5G for cell phones, and parking sensors in vehicles like all the Tesla models or newer Audi models. Improving functionality of these radars will help us improve reception efficiency in electronics and make the world safer to navigate.
